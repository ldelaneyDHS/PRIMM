PRIMM Graphic Organizer

Predict (P)

What do you think the program does?

Run (R)

Run the program and observe its behavior.

What is the program's output?

Investigate (I)

How does the program work?

Identify key concepts and structures used in the code.

Modify (M)

Make a small change to the program and observe the results.

What did you change? What was the result of making that change?

Make (M)
